Pitfall

A 3D platformer built around hand-modelled environments and physics-based traps.

Overview

Pitfall is a 3D platformer where every level is a hand-crafted obstacle course filled with traps, moving platforms, and tight jumps. The project was as much about 3D modelling as it was about game design — every asset in the game was built from scratch in Blender before being imported into Unity. The design challenge was creating levels that felt fair but punishing, with a visual style that made the hazards both beautiful and dangerous.
